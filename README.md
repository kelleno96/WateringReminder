# WateringReminder

An iOS app that helps you keep track of when to water your houseplants.

## Features

- **Plant list** — add and manage all your plants in one place
- **Watering log** — record waterings with a single tap, or log a past watering with a custom date and time
- **Watering history** — view a full history of when each plant was watered
- **Status indicators** — color-coded dots show at a glance whether a plant is doing well (green), due soon (orange), or overdue (red)
- **Push notifications** — enable per-plant reminders on a custom interval (1–365 days) so you never forget to water

## Requirements

- iOS 17+
- Xcode 15+

## Getting Started

1. Clone the repository
2. Open `WateringReminder.xcodeproj` in Xcode
3. Build and run on a simulator or device

## Built With

- [SwiftUI](https://developer.apple.com/xcode/swiftui/) — declarative UI framework
- [SwiftData](https://developer.apple.com/documentation/swiftdata) — persistent storage with automatic schema migration
- [UserNotifications](https://developer.apple.com/documentation/usernotifications) — local push notification scheduling
